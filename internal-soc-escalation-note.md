Internal SOC Escalation Note
Alert Summary

Alert Type: Suspected Microsoft 365 account compromise
Severity: High
Affected Asset: User mailbox / Microsoft 365 account
Status: Escalated for containment review

Summary

A suspected identity compromise was reviewed after a user reportedly interacted with a phishing email. Simulated investigation details included unusual sign-in behavior, suspicious mailbox activity, possible inbox rule abuse, and potential unauthorized forwarding.

Evidence Reviewed
User-reported phishing interaction
Unusual sign-in behavior
Suspicious mailbox rule activity
Possible external forwarding behavior
Need for MFA and active session review
Potential Impact
Unauthorized mailbox access
Credential theft
Hidden or redirected security notifications
Possible exposure of sensitive email content
Continued access through active sessions if not revoked
Recommended Actions
Reset user password
Revoke active sessions
Remove suspicious inbox rules
Disable unauthorized forwarding
Confirm MFA enforcement
Review recent sign-in activity
Search for similar activity across other accounts
Escalation Criteria

Escalate further if additional affected users are identified, suspicious logins continue after containment, mailbox rules reappear, or evidence suggests broader tenant compromise.
